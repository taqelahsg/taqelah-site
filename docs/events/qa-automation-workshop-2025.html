Nov 15, 2025

About This Event

Join us for a comprehensive hands-on workshop covering modern test automation frameworks, CI/CD integration, and best practices for building scalable test suites.

This workshop is designed for QA engineers, developers, and anyone interested in learning how to implement effective test automation strategies in their projects.

What You'll Learn

  • Modern test automation frameworks (Selenium, Playwright, Cypress)
  • CI/CD integration with GitHub Actions and Jenkins
  • Best practices for scalable and maintainable test suites
  • Page Object Model and design patterns
  • Debugging and troubleshooting automated tests
