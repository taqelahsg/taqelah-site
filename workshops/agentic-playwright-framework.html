Workshop Curriculum

Comprehensive 2-day program covering modern test automation and AI-powered testing techniques

Day 1 – Building the Agentic Playwright Framework

  • TypeScript introduction and project setup
  • Variables, data types, and operators
  • Control flow: conditionals, switch statements, and loops
  • Arrays, tuples, objects, and functions
  • Scope management and best practices
  • Prerequisites and environment setup
  • Playwright installation methods and project initialization
  • Writing and executing your first test
  • Code generation with Codegen and advanced options
  • Debugging with TraceViewer and Playwright Inspector
Locator Strategies:
  • Playwright-supported locator strategies and best practices
  • Recording locators with Playwright Inspector and browser plugins
Assertions & Validations:
  • Visibility, presence, and state assertions (enabled/disabled)
  • Text matching, attribute validation, URL and title checks
  • Screenshot comparison and visual testing
UI Elements Interaction:
  • Input fields, buttons, radio buttons, and checkboxes
  • Static and dynamic dropdowns (single and multi-select)
  • Alerts, frames, window handling, drag-and-drop
  • File upload and download handling
  • Screenshot capture and video recording
  • Test parameterization and data-driven testing
  • Reading test data from JSON and CSV files
  • Page Object Model (POM) importance and implementation
  • Access modifiers and base page implementation
  • Playwright fixtures for test setup and teardown
Reporting:
  • Built-in Playwright reporters overview
  • Allure report integration and configuration
API Testing with Playwright:
  • GET, POST, PUT, and DELETE API requests
  • API response validation and assertion strategies
  • Docker introduction and installation
  • Configuring Playwright project for Docker execution
  • Dockerfile and Docker Compose implementation
  • Volume mapping and container orchestration

Day 2 – AI-Powered Testing & Root Cause Analysis

CI/CD Integration:
  • CI/CD fundamentals and pipeline overview
  • Triggering tests with GitHub Actions and Jenkins
Generative AI Integration:
  • Playwright MCP (Model Context Protocol) introduction
  • Automatic test script generation using AI
  • Visual Studio Code + Claude Code integration
  • Creating test scripts using AI-powered coding assistants
  • AI-driven test generation using risk-based analysis
  • Learning from past failures for predictive test selection
  • Generating synthetic test data using LLMs
  • Integrating Promptfoo / Gemini evaluations for test quality
Core Concepts:
  • What is Root Cause Analysis in testing
  • How AI assists RCA across functional, regression, and integration tests
Techniques:
  • Log and screenshot pattern recognition
  • AI-based failure classification (infrastructure vs script vs product issue)
  • Summarizing error traces using LLMs
  • RCA agents that learn from historical bug patterns
Hands-On Exercise:
  • Feed failed test logs into an LLM
  • Generate RCA summary → actionable Jira ticket
  • Evaluate accuracy using Promptfoo metrics
  • Building anomaly-detection models for flaky tests
  • Robust locator strategies and retry logic guided by AI feedback
  • Using natural language generation for release notes & bug summaries
  • Integrating Slack / Confluence bots for auto-reporting
