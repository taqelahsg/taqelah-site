About This Meetup

Join us for an exciting evening of learning and networking with fellow QA professionals! This meetup features two cutting-edge talks on AI-powered automation and modern testing frameworks.

Whether you're a seasoned QA engineer or just starting your journey in test automation, this event will provide valuable insights into the latest tools and techniques shaping the future of quality assurance.

Featured Speakers

Gowthamraj Palani

Staff Quality Engineer at Agoda

Accelerating Test Automation with Playwright MCP and Prompt-Driven Test Generation

Discover how to leverage Playwright's Model Context Protocol (MCP) combined with AI-driven prompt generation to create robust, maintainable test suites. Learn practical strategies for implementing intelligent test automation that adapts to your application's changing needs.

Playwright AI Testing Test Automation

Syam Sasi

Lead Test Engineer at Carousell

Build Your First n8n AI Agent

See a live demonstration of building an AI agent using n8n, the powerful workflow automation tool. Learn how to create intelligent automation workflows that can handle complex testing scenarios, integrate with various tools, and make your QA processes smarter and more efficient.

n8n AI Agents Automation
